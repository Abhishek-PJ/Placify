CC SEE Lab Setup Guide

1. Install Node.js and npm

Install Node.js and npm on your system:
sudo apt install nodejs npm -y
(Alternatively, download from the official Node.js website.)

2. Install AWS Amplify CLI
npm cache clean --force
npm install -g @aws-amplify/cli

3. Create IAM User
Go to AWS IAM Console.

Create a new IAM user with programmatic access.

Assign the necessary permissions.

Download Access Key ID and Secret Access Key.

4. Initialize Amplify
amplify init
Enter project details.
Provide Access Key ID, Secret Access Key, and AWS Region.

5. Add Cognito Authentication
amplify add auth
Choose Default configuration.
Sign-in method: Email.
Complete setup.

6. Deploy Authentication Backend
amplify push

7. Integrate Cognito Auth into React App
npm install aws-amplify @aws-amplify/ui-react

8. Run React Project
npm run dev

9. Create EC2 Instance
Launch an EC2 instance.
Ensure Security Group allows inbound HTTP (port 80) and custom API port (e.g., 4000).

10. Create RDS Database
Launch an RDS MySQL instance.
Make it publicly accessible (or accessible to EC2 security group).

11. Connect EC2 to RDS
On EC2:
git clone https://github.com/Abhishek-PJ/Placify.git
sudo apt update
sudo apt install nodejs npm -y
sudo apt install mysql-client -y
npm install express

12. Create Database Schema
Login to MySQL from EC2:
mysql -h <RDS-endpoint> -u admin -p
Run:
CREATE DATABASE IF NOT EXISTS placify;
USE placify;

CREATE TABLE experiences (
  id INT AUTO_INCREMENT PRIMARY KEY,
  user_id VARCHAR(255),
  company VARCHAR(255),
  role VARCHAR(255),
  description TEXT,
  file_key VARCHAR(512),
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

13. Configure Environment Variables
Go to cd placify/backend and create nano .env and add credentials
DB_HOST=<RDS-endpoint>
DB_USER=placify
DB_PASSWORD=<your-password>
DB_NAME=placify
PORT=4000


14. Run Backend Server
node index.js

15. Test API
Open in browser or Postman:
http://<EC2-public-ip>/api/experiences

16. Enable File Uploads
amplify add storage
amplify push

17. Final Frontend .env
VITE_API_BASE_URL=http://<EC2-public-ip>:4000

18. Grant MySQL Permissions(Only if application is not working)
Login as admin in RDS:
CREATE USER IF NOT EXISTS 'placify'@'%' IDENTIFIED BY 'Jaisriram7$';
GRANT ALL PRIVILEGES ON placify.* TO 'placify'@'%';
FLUSH PRIVILEGES;
